# 📖 [MJA_Bags] <a name="about-project"></a>

> The MJA-BAGS website is a sleek and user-friendly platform designed to showcase stylish bags, enabling customers to explore the collection and connect with the business effortlessly.

**[MJA_Bags]** is a minimal viable product (MVP) designed to:

- Display the range of bags available for purchase.
- Highlight key details about the business.
- Provide a contact option for inquiries or orders.

This is the first version of the website, focusing on simplicity and functionality, with plans for future enhancements.
